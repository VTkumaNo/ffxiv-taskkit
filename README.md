# FINAL FANTASY XIV Task Tool Kit

**非公式のタスク管理ツール**  
FF14（ファイナルファンタジーXIV）の日課・週課・目標タスクを、  
ブラウザ上で簡単に整理・管理できるHTMLアプリです。

> 💡 このツールは非公式ファンメイド作品であり、  
> SQUARE ENIX とは一切関係ありません。

---

## ⚙️ 主な機能
- **デイリー / ウィークリー / 目標タスク** の分類管理  
- ✅ 完了ボタンで進捗を即時反映  
- 🔔 自動保存（ブラウザの localStorage に記録）  
- 📆 **日付・週の変化を検知して自動リセット（v15より）**  
- 💾 バックアップ / JSONインポート機能  
- 🎵 効果音やテーマカラー切替にも対応（Eorzea / Tarkov）

---

## 💾 使い方
1. `index.html` をブラウザで開くだけで動作します。  
   （インストール不要・オフライン対応）  
2. 新規タスクを作成し、「種別」を選択  
   - デイリー：毎日更新される項目  
   - ウィークリー：週単位で更新される項目  
   - 目標：期間指定・長期目標  
3. タスク完了時は「完了」ボタンを押すと自動で保存されます。  
4. 「早見表」タブで一覧表示、「詳細」タブで編集が可能。

---

## 🕒 自動リセットについて（v15）
- **毎日0時** にデイリータスクが未完了へリセット  
- **週が切り替わった時（月曜扱い）** にウィークリーが未完了へリセット  
- 処理は1分ごと・画面復帰時にもチェック  
- 通知メッセージ「デイリー/ウィークリーを自動リセットしました」で確認できます。

---

## 📱 スマホでの利用
- Android: Chrome → 右上メニュー → **ホーム画面に追加**  
- iPhone: Safari → 共有 → **ホーム画面に追加**  
アプリのように全画面起動できます。

---

## ⚠️ 注意事項
- このツールはファンメイドの非公式アプリです。  
  データはすべて **ブラウザの localStorage** に保存されます。  
  URLやブラウザを変えると別データ扱いになります。
